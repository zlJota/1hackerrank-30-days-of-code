**Objetivo**

Hoy profundizaremos en la Herencia (*Inheritance*). Consulta el tutorial adjunto para obtener materiales de aprendizaje y un video instructivo.

**Tarea**

Se te dan dos clases, `Person` y `Student`, donde `Person` es la clase base y `Student` es la clase derivada. El código completado para `Person` y una declaración para `Student` ya se proporcionan en el editor. Observa que `Student` hereda todas las propiedades de `Person`.

Completa la clase `Student` escribiendo lo siguiente:

1. Un constructor de la clase `Student`, el cual tiene $4$ parámetros:
* Un string, `firstName`.
* Un string, `lastName`.
* Un entero, `id`.
* Un arreglo (o vector) de enteros para las calificaciones de los exámenes, `scores`.


2. Un método `calculate()` que devuelve un carácter (`char`), el cual calcula el promedio del objeto `Student` y retorna la letra representativa de dicho promedio según la siguiente escala:

| Promedio ($a$) | Calificación (*Grade*) |
| --- | --- |
| $90 \le a \le 100$ | **O** |
| $80 \le a < 90$ | **E** |
| $70 \le a < 80$ | **A** |
| $55 \le a < 70$ | **P** |
| $40 \le a < 55$ | **D** |
| $a < 40$ | **T** |

**Formato de entrada**

El código predefinido (*stub code*) en el editor lee la entrada y llama al constructor de la clase `Student` con los argumentos necesarios. También llama al método `calculate()`, el cual no recibe argumentos.

La primera línea contiene `firstName`, `lastName` e `id`, separados por un espacio.

La segunda línea contiene el número de calificaciones.

La tercera línea contiene números enteros separados por espacios que describen `scores`.

**Restricciones**

* $1 \le \text{longitud de } firstName, \text{longitud de } lastName \le 10$
* $\text{longitud de } id \equiv 7$
* $0 \le score \le 100$

**Formato de salida**

La salida es gestionada por el código predefinido. Tu salida será correcta si el constructor de la clase `Student` y el método `calculate()` están implementados adecuadamente.

**Entrada de muestra**

```text
Heraldo Memelli 8135627
2
100 80

```

**Salida de muestra**

```text
Name: Memelli, Heraldo
ID: 8135627
Grade: O

```

**Explicación**

Este estudiante tuvo $2$ calificaciones para promediar: $100$ y $80$. El promedio de las calificaciones del estudiante es $\frac{100 + 80}{2} = 90$. Un promedio de $90$ corresponde a la letra de calificación **O**, por lo que el método `calculate()` debe retornar el carácter `'O'`.
