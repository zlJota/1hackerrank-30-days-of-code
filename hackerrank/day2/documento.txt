Aquí está la solución:

python
#!/bin/python3

import math
import os
import random
import re
import sys

#
# Complete the 'solve' function below.
#
# The function accepts following parameters:
#  1. DOUBLE meal_cost
#  2. INTEGER tip_percent
#  3. INTEGER tax_percent
#

def solve(meal_cost, tip_percent, tax_percent):
    tip = meal_cost * (tip_percent / 100)
    tax = meal_cost * (tax_percent / 100)
    total_cost = meal_cost + tip + tax
    print(round(total_cost))

if __name__ == '__main__':
    meal_cost = float(input().strip())

    tip_percent = int(input().strip())

    tax_percent = int(input().strip())

    solve(meal_cost, tip_percent, tax_percent)
Explicación:

Se calcula la propina (tip) multiplicando meal_cost por el porcentaje de propina dividido entre 100.
Se calcula el impuesto (tax) de la misma manera, usando tax_percent.
Se suman meal_cost + tip + tax para obtener el costo total.
Se usa round() para redondear al entero más cercano antes de imprimir.
Verificación con el ejemplo:

meal_cost = 12.00, tip_percent = 20, tax_percent = 8
tip = 12.00 * 0.20 = 2.4
tax = 12.00 * 0.08 = 0.96
total = 12.00 + 2.4 + 0.96 = 15.36
round(15.36) = 15 ✅ Coincide con la salida esperada.

Causas comunes de este error:

Falta la línea de tax o se escribió mal el nombre de la variable (por ejemplo Tax con mayúscula, o tax_percent en vez de tax).
Indentación inconsistente: si mezclaste tabs y espacios, Python a veces no toma la línea como parte del bloque de la función.
